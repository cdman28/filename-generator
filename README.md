# 파일명 수정 도우미 v1.2

정기 보고서 등의 파일명에서 날짜 패턴을 자동으로 감지하고,  
원본 파일을 복사하여 다음 주/월 날짜 이름으로 새 파일을 생성해주는 Windows PC용 앱입니다.

---

## 주요 기능

- **날짜 패턴 자동 감지**: 파일명에서 `260503`, `2026-4월`, `2026년 5월` 등 날짜 패턴을 자동 인식
- **다음 날짜 자동 추론**: 폴더 내 파일들의 날짜 간격을 분석하여 다음 생성 날짜를 자동 계산
- **즐겨찾기 폴더 관리**: 자주 쓰는 보고서 폴더를 즐겨찾기로 등록하여 빠르게 접근- **드래그로 순서 변경** (신규 v1.1): 즉겨찾기 목록에서 마우스 드래그로 순서를 자유롭게 변경
- **주간/월간 그룹 배지** (신규 v1.1): 폴더의 파일 패턴을 분석하여 주간(파란)/월간(보라) 배지 자동 표시- **최근 파일 우선 표시**: 최근 수정된 파일 10개 먼저 표시 (더 불러오기 가능)
- **생성 미리보기**: 파일 생성 전 새 파일명을 미리 확인
- **파일 바로 열기**: 생성 완료 후 결과 창에서 파일을 바로 실행
- **설정 자동저장**: 즐겨찾기 목록은 윈도우 시스템 임시폴더에 자동 저장
- **설정 백업/복원**: 즐겨찾기 설정을 파일로 내보내기/불러오기 지원

---

## 지원 날짜 패턴

| 패턴 | 예시 | 주기 |
|------|------|------|
| `YYMMDD` | `260503` | 주 단위 (7일) |
| `YYYY-N월` | `2026-4월` | 월 단위 |
| `YYYY년 N월` | `2026년 5월` | 월 단위 |

## 실행 방법

### 직접 실행 (Python)
```bash
pip install -r requirements.txt
python main.py
```

### exe 실행
`파일명 수정 도우미 1.2.exe` 를 더블클릭하여 실행합니다.
