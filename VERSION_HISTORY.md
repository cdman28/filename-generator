## v1.3 (2026-05-09)

### 버그 수정

- **즐겨찾기 그룹 자동 감지 오류 수정**: 폴더 추가 시 `FilePatternInfo.cycle` 부재 필드 참조 오류 해결 → `FilePatternInfo.pattern.cycle` 사용
- **일괄 생성 실행 안 되던 문제 수정**: `fi.detected` / `fi.full_path` 등 없는 필드 참조 오류 해결 (`fi.pattern`, `os.path.join(folder, fi.filename)` 사용)

### 기능 개선

- **일괄 생성 미리보기 체크박스 추가**: 일괄 생성 팝업에서 파일별 체크박스 표시, 제외할 파일은 체크 해제 후 생성
- 상단 체크박스로 전체 선택/해제 가능

## v1.2 (2026-05-09)

### 기능 개선

**즐겨찾기 드래그 UX 개선**
- 드래그 중 폴더 이름이 표시된 고스트 창이 커서를 따라다님
- 드롭 위치에 강조 표시 (같은 그룹 내 순서 변경)

**즐겨찾기 트리형 그룹 UI**
- 주간/월간/미분류 섹션 헤더로 폴더 목록을 트리형으로 구분
- 헤더에 폴더 개수 표시 (예: 🗓  주간 (3))
- 주간/월간 헤더에 [일괄 생성 ▶] 버튼 추가: 해당 그룹의 모든 폴더를 날짜 자동 추론 후 한 번에 파일 생성
- 우클릭 메뉴에 그룹 변경(주간↔월간↔미분류) 기능 추가

## v1.0 (2026-04-30)

### 최초 릴리즈

**핵심 기능**
- 파일명 날짜 패턴 자동 감지 (YYMMDD / YYYY-N월 / YYYY년 N월)
- 폴더 내 파일들의 날짜 간격을 분석하여 다음 생성 날짜 자동 추론
- 원본 파일 복사 후 새 날짜 파일명으로 생성

**UI**
- 즐겨찾기 폴더 사이드바 (추가/삭제/이름변경, 우클릭 메뉴)
- 파일 목록 (최근 수정 파일 10개 우선 표시, 더 불러오기)
- 파일별 날짜 패턴 태그 표시 (주 단위 / 월 단위)
- 생성 전 미리보기 팝업 (중복 파일 경고 포함)
- 생성 결과 팝업 (성공 파일 바로 열기 버튼)

**설정 관리**
- 즐겨찾기 목록 윈도우 시스템 임시폴더 자동저장
- 즐겨찾기 설정 파일로 내보내기 / 불러오기 (백업/복원)

**기술 스택**
- Python 3.11+, CustomTkinter 5.2+
- 맑은 고딕 폰트, 흰색 배경 UI (라이트/다크 모드 대응)
